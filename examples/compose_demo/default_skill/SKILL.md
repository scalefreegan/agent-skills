# Code Review Skill

## Overview

This skill helps review code for quality and best practices.

## Instructions

When reviewing code:
1. Check for proper error handling
2. Verify code style consistency
3. Look for potential bugs
4. Suggest improvements

## Default Settings

- Language: Python
